#### Vue是什么
一个 构建用户界面 的 渐进式 框架
* 构建用户界面 ：基于数据动态渲染页面
* 渐进式 ：循序渐进的学习
* 框架 ： 一套完整的项目解决方案，提升开发效率

#### watch监视器
监视数据变化，执行一些业务逻辑或异步请求
* 简单写法：
``` js
watch:{
	数据属性名(newval,oldval){

	},
	'对象属性名'(newval,oldval){

	}
}
```
