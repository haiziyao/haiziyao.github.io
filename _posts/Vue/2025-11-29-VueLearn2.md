		* 全部导入

		``` js
		下载: npm i vant@latest-v2 -S

		//main.js
		import Vant from 'vant'
		import 'vant/lib/index.css'
		Vue.use(Vant)

		<van-button type="info"><van-button/>
		```
		* 自动按需导入

		``` js
		//下载插件
		npm i babel-plugin-import -D

		//在babel.config.js中添加
		module.exports = {
			plugins: [
				['import', {
				libraryName: 'vant',
				libraryDirectory: 'es',
				style: true
				}, 'vant']
			]
		}

		//导入组件
		// 接着你可以在代码中直接引入 Vant 组件
		// 插件会自动将代码转化为方式二中的按需引入形式
		import { Button } from 'vant' 
		Vue.use(Button)
		//一定要记得{}括起来Button
		//我们通常会把导入的代码放入utils/vant-ui.js
		```

##### vw适配
基于postcss插件  实现项目vw适配
* 安装插件 postcss-to-viewport
npm install postcss-px-to-viewport

* 新建postcss.config.js文件，导入配置

	``` js
		module.exports = {
			plugins: {
				'postcss-px-to-viewport' : {
					//标准宽度
					viewportWidth: 375
				}
			}
		}
	```
