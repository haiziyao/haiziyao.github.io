## 初识 Kafka

* zookeeper启动方式
* KRaft启动

### 安装Kafka

``` bash
docker pull apache/kafka:3.7.0
docker run -p 9092:9092 apache/kafka:3.7.0
docker exec -it kafka sh
cd /opt/kafka/bin
./kafka-topic.sh
```

### 使用Kafka

``` bash
./kafka-topic.sh --create --topic hello --bootstrap-server localhost:9092

./kafka-topic.sh --list --bootstrap-server localhost:9092

./kafka-topic.sh --delete --topic hello --bootstrap-server localhost:9092

./kafka-topic.sh --describe --topic hello --bootstrap-server localhost:9092

# alter 只能改参数，topic 名字改不了
./kafka-topic.sh --alter --topic hello --partition 5 --bootstrap-server localhost:9092


./kafka-console-producer.sh --topic quickstart-events --bootstrap-server localhost:9092

./kafka-console-consumer.sh --topic quickstart-events --from-begining --bootstrap-server localhost:9092


/etc/kafka/docker# docker cp f49748080656:/etc/kafka/docker/server.properties ./

    # 配置

    advertised.listeners = PLAINTEXT://ip:9092

docker run --volume /etc/kafka/docker:/mnt/shared/config -p 9092:9092 apache/kafka
```
