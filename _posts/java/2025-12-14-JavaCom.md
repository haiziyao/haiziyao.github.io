* lombok

| 场景               | 是否推荐   | 常用注解                            |
| ---------------- | ------ | ------------------------------- |
| 实体类（Entity / DO） | ✅ 强烈推荐 | `@Data` / `@Getter` / `@Setter` |
| DTO / VO         | ✅ 强烈推荐 | `@Data` / `@Builder`            |
| 构造器注入            | ✅ 强烈推荐 | `@RequiredArgsConstructor`      |
| 日志               | ✅ 推荐   | `@Slf4j`                        |
| 工具类              | ⚠️ 有条件 | `@UtilityClass`                 |
| JPA 实体           | ⚠️ 谨慎  | 避免 `@Data`                      |
| 复杂业务对象           | ❌ 不推荐  | 少用 Lombok                       |
